◆ change_current_frame()

bool change_current_frame ( )

Metoda change_current_frame() zmienia aktualnie wyswietlana klatke i zwraca informacje czy animacja zakonczyla sie.

Metoda change_current_frame() zwieksza aktualnie wyswietlana klatke o 1.

Jesli aktualna klatka jest ostatnia klatka animacji to zwraca 1, w przeciwnym wypadku zwraca 0.

Zwraca

true - animacja zakonczona

false - animacja jeszcze nie zakonczona

Jesli aktualna klatka jest ostatnia klatka animacji to zmienia ja na pierwsza i zwraca true, w przeciwnym wypadku zwraca false

Zwraca

true - animacja zakonczona

return false - animacja nie zakonczona

◆ update()

bool update ( )

Metoda update() aktualizuje animacje i zwraca informacje czy animacja zakonczyla sie.

Metoda update() aktualizuje animacje, zwiekszajac iterator o 1.

Jesli aktualna klatka jest ostatnia klatka animacji to zwraca 1, w przeciwnym wypadku zwraca 0.

Zwraca

true - animacja zakonczona

false - animacja nie zakonczona

Jesli iterator przekroczy predkosc animacji, zwieksza aktualnie wyswietlana klatke o 1. Jesli aktualna klatka jest ostatnia klatka animacji to zmienia ja na pierwsza i zwraca true, w przeciwnym wypadku zwraca false

Zwraca

return true - animacja zakonczona

return false - animacja nie zakonczona
