◆ move()

void move ( )

Przesuniecie preciwnika

Metoda przesuwa przeciwnika w zaleznosci od aktualnego kierunku ruchu.

Jesli kierunek ruchu jest rowny 1 lub - 1, przeciwnik przesuwa sie w lewo lub prawo, unikajac krawedzi ekranu. Jesli kierunek ruchu jest rowny 0, przeciwnik przesuwa sie w dol. Jesli przeciwnik dotrze do konca ekranu, zmienia kierunek ruchu na przeciwny.
